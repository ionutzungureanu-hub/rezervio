Configureaza o singura data pentru sincronizare
1Creeaza proiect Firebase

Mergi la console.firebase.google.com
Add project → nume (ex: pensiunea-mea) → Create

2Activeaza Realtime Database

Build → Realtime Database → Create database
→ europe-west1 → Start in test mode → Enable

3Seteaza regulile (tab Rules)

Sterge tot, lipeste codul de mai jos, apasa Publish:

{"rules":{"reservations":{".read":true,".write":true},"config":{".read":true,".write":true}}}
4Lipeste URL-ul bazei de date

Copiaza URL-ul din Realtime Database (sus)

URL invalid. Trebuie sa contina firebasedatabase.app
